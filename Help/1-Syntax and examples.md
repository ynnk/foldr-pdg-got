# Write graphs 


## Using special properties

### @ Nodes
* `label` displayed node text label 
* `image` url image 
* `shape` node shape in `circle`,`square`,`triangle`,`diamond`,`square`
* `size`   bigger or smaller node default is `1`


### _ Relations
* `label` 
* `weight`
